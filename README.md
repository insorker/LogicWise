# LogicWise

## 简介

- insorker
- 2021/3/25

实现数字电路中的逻辑表达式转化为真值表

## 使用

双击 .exe 文件即可使用

## 说明

**“请输入变量个数：”**

有多少个变量，就输入多少

如：表达式为ABC + BC，则输入3

**“请输入表达式：”**

输入待转换的表达式即可，中间可以插入空格

如：输入 `!A!B + A!B`

> 所有可用符号：
>
> +：或
>
> *：与
>
> !：非
>
> ()：括号
>
> 注意：乱输入肯定是会出错的

**“是否使用卡诺图形式输出真值表？”**

这个挺有意思的

输入 n，则按正常的真值表输出

001

010

011

100

··· ···

如果想画卡诺图，可以输入 y，则输出为（循环码模式）

000

001

011

010

110

··· ···
